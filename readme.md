# Project Description: Expense Management Application
Project Overview

The Expense Management Application is a microservices-based web application designed to manage employee and organizational expenses. The application enables users to create, track, update, and monitor expense records through a secure and scalable platform.

The project demonstrates end-to-end DevOps implementation, including Infrastructure as Code (IaC), CI/CD automation, containerization, Kubernetes orchestration, monitoring, logging, and cloud deployment.

The primary objective is to automate expense management while ensuring high availability, scalability, security, and operational efficiency.

# Tech Stack 

1. AWS Cloud (EC2- 3 Servers)
2. Database (DB)- mysql
3. Backend - nginx
4. Frontend - nodejs
